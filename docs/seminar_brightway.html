Code-along with Brightway

Pre-seminar installation tasks

For this seminar, you should have a working Brightway environment on your computer. For this, you need to:

  1. If not already done, install Python and a Python environment manager,
  2. Install or update Brightway.

Installing Python with Anaconda

If you already have Python installed, skip this step. Otherwise, follow this link to download Anaconda. Follow the installation instructions of the installer.

(Optional - Mamba)

If you think that you will use Python in the future, I recommend installing and using Mamba. It is much faster for “solving environments”, i.e., for installing or updating packages, and for creating new environments. For installing it, open an “Anaconda prompt” and use the command recommended here:

conda install -c conda-forge mamba

Installing Brightway

Open an “Anaconda prompt” (on Windows, you find it on the Start menu) and follow the instructions for installing Brightway.
